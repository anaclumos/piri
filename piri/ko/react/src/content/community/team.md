## React Core {/*react-core*/}

React Core 팀원들은 React DOM과 React Native를 구동하는 엔진, React DevTools, 그리고 React 문서 웹사이트의 핵심 구성 요소 API에 전념하여 풀타임으로 일하고 있습니다.
